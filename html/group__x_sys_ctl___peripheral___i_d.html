Detailed Description

Values that show xSysCtl Peripheral ID.

1. Where to use this group

Values that can be passed to the xSysCtlPeripheralEnable() xSysCtlPeripheralDisable() xSysCtlPeripheralReset() APIs as the ulPeripheral parameter.

2.CoX Port Details
