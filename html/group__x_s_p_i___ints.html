Detailed Description

Values that show xSPI interrupts
.

Values show that xSPI Interrupts Values that can be passed to SPIIntEnable, SPIIntDisable, and SPIIntFlagClear as the ulIntFlags parameter, and returned from SPIIntFlagGet.

1. Where to use this group

Values that can be passed to SPIIntEnable, SPIIntDisable, and SPIIntClear as the ulIntFlags parameter, and returned from SPIIntStatus.

2. CoX Port Details
